      バスにオーディオを乗せる

      デフォルトでは、SuperColliderはオーディオに対して1024のバスを持ちます。バスは配列の中の要素として存在します。

      ////////////////////////////////////////////////////////////////////////////////////////////////////

      // オーディオ・バス（チャンネル）の配列 [ channel0, channel1, channel2, channel3, channel4, ... , ..., ..., etc., ... channel127 ]

      ////////////////////////////////////////////////////////////////////////////////////////////////////

      オーディオ・バスにデータを乗せるには、オーディオ・レートで動作するOutというugenを使います。
