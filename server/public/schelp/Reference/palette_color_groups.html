    Color groups

    A list of color groups within a palette, and their meaning

      A palette (see QPalette) consists of colors assigned to various color roles (see Color roles) within the three color groups. The following is a list of possible symbols to denote color groups, and their meanings.
      GroupMeaning
      'active'Containes colors used by views on the active window.
      'inactive'Containes colors used by views on inactive windows.
      'disabled'Containes colors used by disabled views.
      'normal'A synonym for 'active'.
