    MIDIFuncBothCAMessageMatcher : AbstractMessageMatcher : Object

    Matches incoming MIDI messages to responder funcs based on message channel and source
    Source: ResponseDefs.sc
    See also: AbstractMessageMatcher, MIDIFuncSrcMessageMatcher, MIDIFuncChanMessageMatcher, MIDIFuncChanArrayMessageMatcher, MIDIFuncSrcMessageMatcherNV, MIDIFuncBothMessageMatcher

    Description

    This is used by MIDIMessageDispatcher to match incoming MIDI messages to instances of MIDIFunc or MIDIdef using message source and MIDI channel (when the MIDIFunc specifies an array of channels). This class is private, and generally users should not need to address instances directly.
