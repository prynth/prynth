      LinCongL : LinCongN : ChaosGen : UGen : AbstractFunction : Object

      Linear congruential chaotic generator
      Source: Chaos.sc
      See also: LinCongC, LinCongN

      Description

      A linear-interpolating sound generator based on the difference equation:

              x(n+1) = (a * x(n) + c) % m

      The output signal is automatically scaled to a range of [-1, 1].
