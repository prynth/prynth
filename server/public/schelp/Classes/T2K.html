      T2K : A2K : PureUGen : UGen : AbstractFunction : Object

      Audio rate trigger to control rate trigger converter
      Source: Line.sc
      See also: T2A, K2A, A2K

      Description

      Converts audio rate trigger into control rate trigger, using the maximum trigger in the input during each control period.

      Class Methods

      T2K.kr(in: 0)

      From superclass: A2K

      Arguments:

      in

      input signal.

      Examples

      // this does not work:
(
{
    var trig = Dust.ar(4);
    Trig.kr(trig, 0.1) * SinOsc.ar(800) * 0.1
}.play;
)

// this works:
(
{
    var trig = T2K.kr(Dust.ar(4));
    Trig.kr(trig, 0.1) * SinOsc.ar(800) * 0.1
}.play;
)
