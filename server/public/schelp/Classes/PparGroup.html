      PparGroup : PAbstractGroup : FilterPattern : Pattern : AbstractFunction : Object

      Starts a new ParGroup and plays the pattern in this group
      Source: Pfx.sc
      See also: ParGroup, Pgroup

      Description

      The class has a semantics similar to Pgroup, but instead of a Group, it creates a ParGroup on the server.
