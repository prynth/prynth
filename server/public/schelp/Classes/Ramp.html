    Ramp

    Break a continuous signal into line segments

        Description

        Break a continuous signal into linearly interpolated segments with specific durations.

        Feeding Ramp with noise is similar to LFNoise1

        Ramp.kr(WhiteNoise.kr(1),0.5)

        is equal to:

        LFNoise1.kr(1 / 0.5)

        For smoothing out control signals, take a look at Lag and VarLag

        Class Methods

        *ar (in: 0, lagTime: 0.1, mul: 1, add: 0)

        From superclass: Lag

        *kr (in: 0, lagTime: 0.1, mul: 1, add: 0)

        From superclass: Lag

        Arguments:

        in

        The input signal.

        lagTime

        segment duration in seconds.

        mul

        Output will be multiplied by this value.

        add

        This value will be added to the output.

        Examples

        s.boot;
(
// used to lag pitch
{
    SinOsc.ar(        // sine wave
        Ramp.kr(            // lag the modulator
            LFPulse.kr(4, 0, 0.5, 50, 400),    // frequency modulator
            Line.kr(0, 1, 15)                // modulate lag time
        ),
        0,    // phase
        0.3    // sine amplitude
    )
}.scope;
)
