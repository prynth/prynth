      BHiPass
      12dB/oct rolloff - 2nd order resonant Hi Pass Filter
      BHiPass4

      BLowShelf

      BHiShelf

      BBandPass

      BBandStop

      The B equalization suite is based on the Second Order Section (SOS) biquad UGen.
