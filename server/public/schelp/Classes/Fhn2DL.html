    FitzHughNagumo Neuron Firing Oscillator

        Description

        du/dt= rateu*(u-0.3333*u^3 - w)
dw/dt= ratew*(b0+b1*u - w)
        NOTE: All inputs can have .kr rate UGens plugged in.
