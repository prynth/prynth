    SCDocNode : Object

    An SCDoc parsed document node
    Source: SCDoc.sc
    See also: SCDoc

    Description

    This class is used internally by SCDoc to represent a node in the parsed document tree returned by the parser. It has an id symbol, optional text and optional children.
