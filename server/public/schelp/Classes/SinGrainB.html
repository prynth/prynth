    SinGrainB : JoshGrain : UGen : AbstractFunction : Object
    Extension

    Granular synthesis with sinusoidal grains and user supplied envelope
    Source: GrainSynths.sc
    See also: SinGrain, SinGrainI, SinGrainBF, SinGrainBBF, SinGrainIBF

    Description

    Class Methods

    SinGrainB.ar(trigger: 0, dur: 1, freq: 440, envbuf, mul: 1, add: 0)

    Arguments:

    trigger

    A kr or ar trigger to start a new grain. If ar, grains after the start of the synth are sample accurate.

    dur

    Size of the grain. Polled at creation time.

    freq

    The freq of the grain generators internal oscillator. Polled at creation time.

    envbuf

    A buffer with a stored signal to be used for the envelope of the grain.

    mul
    add
