      Hasher : UGen : AbstractFunction : Object

      Scrambled value with a hash function.
      Source: Noise.sc

      Description

      Returns a unique output value from -1 to +1 for each input value according to a hash function. The same input value will always produce the same output value. The input need not be in the range -1 to +1.

      Class Methods

      Hasher.ar(in: 0.0, mul: 1.0, add: 0.0)

      Hasher.kr(in: 0.0, mul: 1.0, add: 0.0)

      Arguments:

      in

      The input signal.

      mul

      Output will be multiplied by this value.

      add

      This value will be added to the output.
