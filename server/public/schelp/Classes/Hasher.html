    Randomized value.

        Description

        Returns a unique output value from zero to one for each input value according to a hash function. The same input value will always produce the same output value. The input need not be from 0 to 1.

        Class Methods

        *ar (in: 0, mul: 1, add: 0)

        *kr (in: 0, mul: 1, add: 0)

        Arguments:

        in

        The input signal.

        mul

        Output will be multiplied by this value.

        add

        This value will be added to the output.
