        Description

        Generates linearly interpolated random values at a rate given by the nearest integer division of the sample rate by the freq argument.

        Class Methods

        *ar (freq: 500, mul: 1, add: 0)

        From superclass: LFNoise0

        *kr (freq: 500, mul: 1, add: 0)

        From superclass: LFNoise0

        Arguments:

        freq

        Approximate rate at which to generate random values.

        mul

        Output will be multiplied by this value.

        add

        This value will be added to the output.

        { LFNoise1.ar(1000, 0.25) }.play;

// modulate frequency

{ LFNoise1.ar(XLine.kr(1000, 10000, 10), 0.25) }.play;

// as frequency modulator

{ SinOsc.ar(

        0, 0.2
    )
