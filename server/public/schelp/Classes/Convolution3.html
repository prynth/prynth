    Convolution3 : UGen : AbstractFunction : Object

    Time based convolver.
    Source: FFT2.sc
    See also: Convolution, Convolution2, Convolution2L

    Description

    Strict convolution with fixed kernel which can be updated using a trigger signal. The convolution is performed in the time domain.

    NOTE: Doing convolution in time domain is highly inefficient, and probably only useful for either very short kernel sizes, or for control rate signals. See Convolution2 and Convolution2L for more efficient convolution UGens.
