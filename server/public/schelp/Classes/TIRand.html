    TIRand

    Triggered integer random number generator.
    Source: /usr/local/share/SuperCollider/SCClassLibrary/Common/Audio/Noise.sc

        Description

        Generates a random integer value in uniform distribution from lo to hi each time the trigger signal changes from nonpositive to positive values.

        Class Methods

        *ar (lo: 0, hi: 127, trig: 0)

        *kr (lo: 0, hi: 127, trig: 0)

        Arguments:

        lo

        Lower limit of the output range.

        hi

        Upper limit of the output range.

        trig

        The trigger. Trigger can be any signal. A trigger happens when the signal changes from non-positive to positive.
