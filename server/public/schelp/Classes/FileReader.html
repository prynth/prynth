      FileReader : Stream : AbstractFunction : Object

      file reader for space-delimited text files
      Source: TabFileReader.sc
      Subclasses: CSVFileReader, SemiColonFileReader, TabFileReader
      See also: File

      Description

      FileReader reads space-delimited text files into 2D arrays line by line.

      For tab delimited files use TabFileReader. For semi-colon-delimited files use SemiColonFileReader. For comma-separated files use CSVFileReader.
