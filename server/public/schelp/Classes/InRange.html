    InRange

    Tests if a signal is within a given range.
    Source: /usr/local/share/SuperCollider/SCClassLibrary/Common/Audio/Trig.sc
    Inherits from: UGen : AbstractFunction : Object
    Subclasses: Clip, Fold, Schmidt, Wrap

        Description

        If in is ≥ lo and ≤ hi output 1.0, otherwise output 0.0. Output is initially zero.

        Class Methods

        *ar (in: 0, lo: 0, hi: 1)

        *kr (in: 0, lo: 0, hi: 1)

        *ir (in: 0, lo: 0, hi: 1)

        Arguments:

        in

        Signal to be tested.

        lo

        Low threshold.

        hi

        High threshold.
