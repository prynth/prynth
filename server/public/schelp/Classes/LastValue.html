        Description

        Output the last value before the input changed more than a threshhold.

        Class Methods

        *ar (in: 0, diff: 0.01)

        *kr (in: 0, diff: 0.01)

        Arguments:

        in

        The input signal.

        diff

        Difference threshhold.

        Examples

        d = { arg freq=440; SinOsc.ar(LastValue.ar(freq, 20), 0, 0.2) }.play;

d.set(\freq, 400);
d.set(\freq, 200);
d.set(\freq, 670);
d.set(\freq, 680);
d.set(\freq, 695);
d.free;
