      LFTri : LFSaw : PureUGen : UGen : AbstractFunction : Object

      Triangle oscillator
      Source: Osc.sc
      See also: LFCub, LFPar, LFPulse, LFSaw

      Description

      A non-band-limited triangle oscillator. Output ranges from -1 to +1.

      Class Methods

      LFTri.ar(freq: 440, iphase: 0, mul: 1, add: 0)

      From superclass: LFSaw

      LFTri.kr(freq: 440, iphase: 0, mul: 1, add: 0)

      From superclass: LFSaw

      Arguments:

      freq

      Frequency in Hertz.

      iphase

      Initial phase offset. For efficiency reasons this is a value ranging from 0 to 4.

      mul

      Output will be multiplied by this value.

      add

      This value will be added to the output.

      // plot:
{ LFTri.ar(40) }.plot(0.1);
{ LFTri.ar(XLine.kr(1, 800, 0.1)) }.plot(0.1);

{ LFTri.ar(500, 0, 0.1) }.play
