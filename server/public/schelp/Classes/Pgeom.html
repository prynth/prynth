      Pgeom : Pattern : AbstractFunction : Object

      geometric series pattern
      Source: Patterns.sc
      See also: Pseries

      Description

      Returns a stream that behaves like a geometric series.

      Class Methods

      Arguments:

      start

      start value.

      grow

      multiplication factor.
