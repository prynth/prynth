        Description

        Allows input signal value to pass when gate is positive, otherwise holds last value.

        Class Methods

        *ar (in: 0, trig: 0)

        From superclass: Latch

        *kr (in: 0, trig: 0)

        From superclass: Latch

        Arguments:

        in

        The input signal.

        trig

        Gate - can be any signal. The output is held fixed when this is non-positive.
