    FlowVar

    A place holder, which when accessed pauses a thread until the place holder has a value
    Source: /usr/local/share/SuperCollider/SCClassLibrary/Common/Core/Condition.sc
    Inherits from: Object
    See also: Ref, Thunk, Condition, Routine

      Description

      (
a = FlowVar.new;
fork {
    "I am waiting, please enter a value ...".postln;
    a.value.postln;
    "...... ok, done.".postln;
}
)

// later, set the value. Then we can continue ...
a.value = 2;

      Class Methods

      *new (inVal: 'unbound')

      Return a new instance,

      Arguments:

      inVal

      If a value is given here, the FlowVar will not block execution.

      Instance Methods

      -value

      -value = inVal

      Set the value bound to the FlowVar.

      The getter returns the value bound to the FlowVar. If the value is not yet available, hold execution (this requires thie method to be called from within a Routine or similar thread.).

      Arguments:

      inVal

      Any object.
