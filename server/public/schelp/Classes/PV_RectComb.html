        Description

        Makes a series of gaps in a spectrum.

        Class Methods

        *new (buffer, numTeeth: 0, phase: 0, width: 0.5)

        Arguments:

        buffer

        FFT buffer.

        numTeeth

        Number of teeth in the comb.

        phase

        Starting phase of comb pulse.

        width

        Pulse width of the comb.
