        Description

        Internally used by Server.

        Class Methods

        *new (server, startBus: 0, numChans, min: -90, max: 6, persist: false)

        Create and return a new instance of Volume for a given server, ranging from startBus over numChans (usually the server's number of output bus channels).

        Arguments:

        server

        a server

        startBus

        start bus

        numChans

        number of channels

        min

        minimum volume in decibel

        max

        minimum volume in decibel

        persist

        whether to persist a reset

        Inherited class methods

        Instance Methods

        -play: METHOD NOT FOUND!

        run the synth that controls the volume. If mute is true, mute the output. (default: false)

        -free

        From superclass: Object

        free the synth that controls the volume.

        -mute

        mute output

        -unmute

        unmute output

        -volume

        -volume = aVolume

        set the volume (in db)

        -lag

        -lag = aLagTime

        set the lag time that dampens volume changes

        -setVolumeRange (argMin, argMax)

        set the volume range

        -gui (window, bounds)

        create a volume gui.
