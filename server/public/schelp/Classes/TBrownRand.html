      TBrownRand : UGen : AbstractFunction : Object
      Extension

      triggered random walk generator
      Source: bhobGens.sc
      See also: TRand, TBetaRand, TGaussRand

      Description

      Generates a random walk from lo to hi with a Gendyn distribution each time the trig signal changes from nonpositive to positive values.

      Class Methods

      TBrownRand.ar(lo: 0, hi: 1.0, dev: 1.0, dist: 0, trig: 0, mul: 1.0, add: 0.0)

      TBrownRand.kr(lo: 0, hi: 1.0, dev: 1.0, dist: 0, trig: 0, mul: 1.0, add: 0.0)

      Arguments:

      lo

      minimum value

      hi

      maximum value

      dev

      maximum step (0-1)

      dist

      Gendyn distribution (0-5: see Gendy1 for details)

      trig

      trigger signal
