      WAmp : UGen : AbstractFunction : Object
      Extension

      windowed amplitude follower
      Source: BatUGens.sc
      See also: Amplitude, TrigAvg

      Description

      Averages and outputs the absolute value of incoming signals received between now and (now - winSize) seconds.

      Class Methods

      WAmp.kr(in: 0.0, winSize: 0.1)

      Arguments:

      in

      Signal input.

      winSize

      The window size in seconds. Not modulatable.
