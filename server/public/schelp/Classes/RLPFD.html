        Description

        This filter can be a little quiet, so i often multiply it by 3 or so. Watch out for high frequencies + hi res + hi dist = feedback.

        Class Methods

        *ar (in, ffreq: 440, res: 0, dist: 0, mul: 1, add: 0)

        *kr (in, ffreq: 440, res: 0, dist: 0, mul: 1, add: 0)

        Arguments:

        in

        audio input

        ffreq

        cutoff freq

        res

        resonance (0-1)

        dist

        distortion (0-1). interacts with resonance
