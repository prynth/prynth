        Description

        Demand rate geometric series UGen.

        See Pgeom for structurally related equivalent.

        Class Methods

        *new (start: 1, grow: 2, length: inf)

        Arguments:

        start

        Start value. Can be a number or any other UGen.

        grow

        Value by which to grow (x = x[-1] * grow). Can be a number or any other UGen.

        length

        Number of values to create.

        Discussion:

        The arguments can be a number or any other ugen
