      Pmulpre : Psetpre : FilterPattern : Pattern : AbstractFunction : Object

      multiplies with value of a key in event stream, before it is passed up
      Source: FilterPatterns.sc
      See also: Pmul, Pmulp

      Description

      Multiplies with a value in an event, before it is passed up the stream. To multiply with the value after it has been passed down, use Pmul.

      Pmulpre does not override incoming values:

      Class Methods

      Pmulpre.new(name, value, pattern)

      From superclass: Psetpre

      Arguments:

      name
      value

      can be a pattern or a stream. The resulting stream ends when that incoming stream ends.
