      Integrator : Filter : PureUGen : UGen : AbstractFunction : Object

      A leaky integrator.
      Source: Filter.sc

      Description

      Integrates an input signal with a leak. The formula implemented is:

      out(0) = in(0) + (coef * out(-1))

      Class Methods

      Integrator.ar(in: 0, coef: 1, mul: 1, add: 0)

      Integrator.kr(in: 0, coef: 1, mul: 1, add: 0)

      Arguments:

      in

      The input signal.

      coef

      Leak coefficient.

      mul

      Output will be multiplied by this value.

      add

      This value will be added to the output.
