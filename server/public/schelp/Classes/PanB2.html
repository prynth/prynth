      PanB2 : MultiOutUGen : UGen : AbstractFunction : Object

      2D Ambisonic B-format panner.
      Source: Pan.sc
      See also: BiPanB2, DecodeB2, PanB, Rotate2

      Description

      Encodes a mono signal to 2-dimensional ambisonic B-format.

      Class Methods

      PanB2.ar(in, azimuth: 0, gain: 1)

      PanB2.kr(in, azimuth: 0, gain: 1)

      Arguments:

      in

      The input signal.

      azimuth

      Position around the circle from -1 to +1. -1 is behind, -0.5 is left, 0 is forward, +0.5 is right, +1 is behind.

      gain

      Amplitude control.
