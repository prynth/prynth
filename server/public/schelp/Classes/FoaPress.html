        Description

        Apply press to a first order ambisonic signal (B-format) along an arbitrary axis.

        NOTE: FoaPress is usually called via the convenience wrapper FoaTransform.

        Class Methods

        *ar (in, angle: 0, theta: 0, phi: 0, mul: 1, add: 0)

        Arguments:

        in

        The B-format signal, an array: [w, x, y, z]

        angle

        The distortion angle, in radians. -pi/2 to pi/2

        theta

        Azimuth, in radians.

        phi

        Elevation, in radians.

        mul

        Output will be multiplied by this value.

        add

        This value will be added to the output.

        Discussion:

        Applies press along the axis defined by theta and phi. See FoaPressX.

        NOTE: Corresponding matrix transformer: FoaXformerMatrix: *newPress
