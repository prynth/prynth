        Description

        Delays a trigger by a given time. Any triggers which arrive in the time between an input trigger and its delayed output, are ignored.

        Class Methods

        *ar (in: 0, dur: 0.1)

        From superclass: Trig1

        *kr (in: 0, dur: 0.1)

        From superclass: Trig1

        Arguments:

        in

        Input trigger signal.

        dur

        Delay time in seconds.
