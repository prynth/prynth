      Line : UGen : AbstractFunction : Object

      Line generator.
      Source: Line.sc
      See also: XLine

      Description

      Generates a line from the start value to the end value.

      Class Methods

      Line.ar(start: 0, end: 1, dur: 1, mul: 1, add: 0, doneAction: 0)

      Line.kr(start: 0, end: 1, dur: 1, mul: 1, add: 0, doneAction: 0)

      Arguments:

      start

      Starting value.

      end

      Ending value.

      dur

      Duration in seconds.

      mul

      Output will be multiplied by this value.

      add

      This value will be added to the output.

      doneAction

      A doneAction to be evaluated when the Line is completed. See
